-- app-switcher is used to show app-switcher component to switch between apps.
Package Name:
fastn-community.github.io/app-switcher

Overview

The app-switcher component is already included inside doc-site. If you are using doc-site then don't add app-switcher as fastn dependency.

How to use?

Add below code to your FASTN.ftd and Import app-switcher into your .ftd file
-- fastn.dependency: fastn-community.github.io/app-switcher
Lang:
ftd
-- import: fastn-community.github.io/app-switcher

-- app-switcher.app-switcher:
apps: $apps
Lang:
ftd

Properties:

Below are details of all properties of this component.
apps

apps property type is fastn.app-ui it need fastn.app list to show apps in its app container.

Add all your apps into your FASTN.ftd file.

Sample app code:
-- fastn.app: Movie DB
mount-point: /todos/
package: kameri-app.herokuapp.com
end-point: https://kameri-service.herokuapp.com
readers: fastn-fans
config: db_url=$ENV.DB_URL
Lang:
ftd
$is-open
$is-open property is a boolean type property by default its false. If sets true by default app-switcher will be shown on your fastn pages.
$current-app
$current-app property property type is integer. It makes app active which in a list is selected.
